🧠 About the Project:
This is an interactive Machine Learning web application that predicts sales based on advertising budget (TV, Radio, Newspaper).
It helps businesses understand how marketing investments impact sales performance 📊

✨ Features:
🎯 Single Sales Prediction (manual input)
📂 Bulk CSV Upload Prediction
📊 Interactive Charts (Bar / Scatter / Gauge)
📉 Business Insight Labels (Low / Medium / High ROI)
📥 Download Prediction Reports
⚡ Real-time ML inference
