# Task 3: Car Price Prediction using Machine Learning

## Objective
To predict car prices based on various features using machine learning techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Algorithm
- Linear Regression

## Steps Performed
- Data preprocessing
- Feature encoding
- Model training
- Model evaluation
- Visualization of results

## How to Run
1. Place `car_price_prediction.py` and `car_data.csv` in the same folder
2. Run:
   python car_price_prediction.py
